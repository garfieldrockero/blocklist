# AdGuard Blocklist

Custom blocklist for AdGuard Home.

## Usage

Add this URL in AdGuard Home:

```
https://raw.githubusercontent.com/garfieldrockero/blocklist/main/blocklist.txt
```

**Steps:**
1. AdGuard Home → Filters → DNS blocklists
2. Add blocklist → Add a custom list
3. Paste the URL and save

## Format

```
||domain.com^
||ads.example.com^
```
